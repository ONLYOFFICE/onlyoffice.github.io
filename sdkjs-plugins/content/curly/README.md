# Curly

Turn any document into a reusable template: mark the parts that change with
`{tags}` and fill them from the task pane — a finished, ready-to-send copy in
seconds.

Curly is built for documents you send out again and again — contracts,
engagement letters, quotes, invoices, certificates — where only a handful of
details change each time.

## How it works

1. Open a document you'd like to reuse and wrap the parts that change in
   curly braces, e.g. `Dear {Client name},` — that's the whole "template
   format".
2. Click **Curly** in the **Plugins** tab. The pane scans the document and
   lists every tag it finds.
3. Type the values (or pick from your saved sets), and Curly produces a
   filled-in copy — the original template stays untouched.

The pane's interface follows your editor language automatically.

## Installation

- **Plugin Manager (recommended):** open the **Plugins** tab → **Plugin
  Manager**, find *Curly* under Marketplace and click **Install**.
- **Manual / self-hosted Document Server:** download the plugin from
  [curly.io/install/onlyoffice](https://curly.io/install/onlyoffice) and
  follow the instructions there — the same page covers ONLYOFFICE Desktop,
  Document Server, and compatible forks.
