# NLP Solver for OnlyOffice

A nonlinear programming (NLP) solver plugin for OnlyOffice Spreadsheet. Define a target cell, set side conditions, and specify parameter cells with optional boundsâ€”then let the solver find optimal values to minimize your target function.

## Features

- Customized Nelder-Mead (simplex downhill) implementation
- Handles nonlinear problems with nonlinear side conditions
- WebAssembly backend written in Go for parallel, high-performance calculations
- Parameter bounds with optional strict enforcement
- Configurable iteration count
