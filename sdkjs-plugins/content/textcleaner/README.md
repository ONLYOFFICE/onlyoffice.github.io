# Text Cleaner

A smart text cleaning and formatting plugin for ONLYOFFICE Document Editor that helps you clean and standardize text formatting with advanced options.

## Overview

Text Cleaner is an intelligent plugin that goes beyond simple "clear formatting" functionality. It provides granular control over text formatting while preserving important styling elements like bold and italic text. The plugin offers comprehensive text cleaning capabilities including:

- **Smart Formatting Removal**: Remove specific formatting elements while preserving others
- **Font Standardization**: Apply consistent font families and sizes across your document
- **Text Case Conversion**: Multiple case conversion options (sentence case, uppercase, lowercase, etc.)
- **Special Formatting Control**: Manage caps, small caps, letter spacing, and vertical alignment
- **Context Menu Integration**: Quick access to cleaning functions through right-click menu

## Usage

### Basic Usage

1. **Open the Plugin**:
   - Go to **Plugins** ‚Üí **Text Cleaner**
   - Or use the context menu: Right-click ‚Üí **Text Cleaner**

2. **Select Text** (Optional):
   - Select specific text to clean, or
   - Leave no selection to clean the entire document

3. **Configure Options**:
   - Expand the sections you want to use
   - Check/uncheck specific formatting options
   - Choose font family and size for standardization
   - Select text case conversion option

4. **Apply Cleaning**:
   - Click the **Clean** button
   - Wait for the processing to complete
   - Review the results

### Advanced Usage

#### Context Menu Integration
- Right-click on selected text
- Choose **Text Cleaner** from the context menu
- Select specific cleaning options
- Apply changes instantly

#### Batch Processing
- Use "Select All" option to apply all cleaning functions
- Process entire documents with consistent formatting
- Use undo functionality to revert changes if needed

#### Font Standardization
- Choose from popular font families (Arial, Calibri, Times New Roman, etc.)
- Select appropriate font sizes for your document type
- Apply consistent typography across the entire document

### Tips and Best Practices

1. **Backup Your Document**: Always save a backup before applying extensive formatting changes
2. **Test on Small Sections**: Try the plugin on a small text selection first
3. **Use Undo Feature**: The plugin provides undo functionality to revert changes
4. **Combine Options**: Mix and match different cleaning options for optimal results
5. **Font Consistency**: Use font standardization for professional documents
